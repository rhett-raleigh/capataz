# Notes

Lightweight project knowledge that lives alongside the code.

```
notes/
  decisions/   ← ADRs: why we chose X over Y
  ideas/       ← rough captures, not yet committed to
  projects/    ← work breakdowns for larger efforts
```

Each folder has a `_template.md`. Copy it to start a new entry.

## Conventions

- **Decisions** are numbered (`001-…`, `002-…`). Use the next available number.
- **Ideas** and **projects** use descriptive filenames (`offline-mode.md`).
- Statuses are freeform but each template suggests a set.
- Git history is the changelog — no need to maintain one separately.
