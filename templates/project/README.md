# Project knowledge template

Portable scaffold for tracking decisions, ideas, and work breakdowns in any
repo. Copy this into a repo's root as `project/` to get started.

```bash
cp -r templates/project/ <your-repo>/project/
```

## What's included

```
decisions/
  _template.md   ← ADR format for recording decisions
ideas/
  _template.md   ← lightweight idea capture
projects/
  _template.md   ← work breakdown for larger efforts
```

See `project/README.md` in this repo for conventions and usage.
